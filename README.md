# Multi-disiplinary Portfolio Website

This project is a modern and responsive web page that introduces students to different career paths in Information Technology (IT). It is designed with a clean user interface and includes a dark/light theme toggle for better user experience.

The goal of this project is to help beginners and students understand various areas in tech such as Web Development, Cybersecurity, Data Science, Artificial Intelligence, and more.

## Highlights

- Dark and Light Mode Toggle.
- Fully Responsive Design (works on mobile, tablet, and desktop).
- Card-based layout for different IT careers.
 Modern UI with smooth hover effects.

## Tech used

- HTML5 – Structure of the web page.
- CSS3 – Styling, layout, and responsiveness.
- JavaScript – Theme toggle functionality.
- Google Fonts – Modern typography.

## Pages used

- `index.html`: Home page with hero carousel, skills, and projects.
- `about.html`: About section with profile image and description.
- `contact.html`: Contact form and contact details.

   |-- style.css

## Running Locally

Open `index.html` in your browser. No build step required.

## How it works

- Navbar: Contains the title and theme toggle button.
- Hero Section: Introduces the purpose of the page.
- Career Cards Section: Displays different IT fields.
- Footer: Contains basic project information.

## How to run

- Open the project folder in VS Code.
- Locate the index.html file.
- Right-click and select "Open with Live Server" (or double-click the file).
- View the project in your browser.
